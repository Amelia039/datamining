Pengertian Diskritisasi#

Merupakan proses mengubah data numerik yang bersifat kontinu menjadi data kategorik yang bersifat diskrit dengan mengelompokkan setiap data ke dalam interval-interval tertentu. Terdapat 2 pendekatan:

  • Equal-width Intervals Equal-width intervals adalah discretization yang membagi data numerik menjadi beberapa kelompok dengan lebar kelompok yang kurang lebih sama besar. Dilakukan dengan membagi-bagi data ke dalam N buah interval (bins) dengan lebar (width) yang seragam antar interval.

  • Equal-frequency Intervals Equal-frequency intervals adalah discretization yang membagi data numerik menjadi beberapa kelompok dengan jumlah anggota yang kurang lebih sama besar.

#Proses Diskritisasi dengan Basis Entropy

Merupakan jenis algoritma diskritisasi terawasi yang menggunakan mekanisme top- down.

Tujuan dari algoritma ini adalah mendapatkan partisi yang mengandung baris data dari kelas yang sama sebanyak mungkin.

Entropi pada umumnya digunakan untuk mengukur ketidakpastian dari sekumpulan atribut dari suatu set data. Ukuran entropi dianggap sebagai ukuran ketidakpastian dimana semakin tinggi entropy suatu atribut maka semakin tinggi ketidakpastian

  • Rumus Entropy

\[ \operatorname{Entropy}\left(D_1\right)=-\sum_{i=1}^m P i \log _2 P i \]

  • Rumus Gain

\[ \operatorname{Gain}\left(E_{\text {new }}\right)=E_{\text {initial }}-E_{\text {new }} \]

  • Rumus Info $\( \operatorname{Info}_A(D)=\frac{\left|D_1\right|}{|D|} \operatorname{Entropy}\left(D_1\right)+\frac{\left|D_2\right|}{|D|} \operatorname{Entropy}\left(D_2\right) \)$

Membuat fungsi GetOverollCategory (keseluruhan)#

Digunakan untuk menghitung data keseluruhan yang nantinya digunakan untuk menghitung entropy

def GetOverollCategory(col):
  group = sepal_length.loc[:, :].groupby("kategori").count()
  a = group.loc['A', col]
  b = group.loc["B", col]
  return (a,b,a+b)

Memasukan fungsi splitter#

Digunakan untuk membuat split antara value yang telah ditentukan lalu mengembalikan data yang telah dipisahkan

def splitter(value, col):
  #get data less and greater from value
  less = sepal_length[sepal_length[col] <= value]
  greater = sepal_length[sepal_length[col] > value]

  #calculate into category for each data
  less_group = less.loc[:, :].groupby("kategori").count()
  greater_group = greater.loc[:, :].groupby("kategori").count()

  #get value based on category
  less_category_A = less_group.loc['A',col]
  less_category_B = less_group.loc["B",col]
  greater_category_A = greater_group.loc['A',col]
  greater_category_B = greater_group.loc["B",col]

  return (
      [less_category_A, less_category_B, less_category_A + less_category_B],
      [greater_category_A, greater_category_B, greater_category_A + greater_category_B]
  )

Membuat fungsi entropy#

Untuk melakukan perhitungan entropy (menggunakan rumus entropy)

def entropy (d):
  r1 = (d[0] / d[2]) * np.log2(d[0] / d[2])
  r2 = (d[1] / d[2]) * np.log2(d[1] / d[2])
  return np.sum([r1, r2]) * -1

Membuat fungsi info#

def info(d):
  r1 = (d[0][2] / sepal_length.shape[0]) * entropy(d[0])
  r2 = (d[1][2] / sepal_length.shape[0]) * entropy(d[1])
  return r1 + r2

Membuat fungsi gain#

def gain(Einitial, Enew):
  return Einitial - Enew

Memasukan fungsi splitter#

Digunakan untuk membuat split antara value yang telah ditentukan lalu mengembalikan data yang telah dipisahkan

def splitter(value, col):
  #get data less and greater from value
  less = petal_length[petal_length[col] <= value]
  greater = petal_length[petal_length[col] > value]

  #calculate into category for each data
  less_group = less.loc[:, :].groupby("kategori").count()
  greater_group = greater.loc[:, :].groupby("kategori").count()

  #get value based on category
  less_category_A = less_group.loc['A',col]
  less_category_B = less_group.loc["B",col]
  greater_category_A = greater_group.loc['A',col]
  greater_category_B = greater_group.loc["B",col]

  return (
      [less_category_A, less_category_B, less_category_A + less_category_B],
      [greater_category_A, greater_category_B, greater_category_A + greater_category_B]
  )

Membuat fungsi entropy#

Untuk melakukan perhitungan entropy (menggunakan rumus entropy)

def entropy (d):
  r1 = (d[0] / d[2]) * np.log2(d[0] / d[2])
  r2 = (d[1] / d[2]) * np.log2(d[1] / d[2])
  return np.sum([r1, r2]) * -1

  r2 = (d[1][2] / petal_length.shape[0]) * entropy(d[1])
  return r1 + r2
